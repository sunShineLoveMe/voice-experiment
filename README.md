# 声音传播实验 MVP（Three.js + Vite）

一个用于课堂演示的 3D 教学项目，直观展示声音传播的宏观与微观过程：

- 宏观层：喇叭鼓膜振动、波前逐层推进、小球依次响应且振幅衰减
- 微观层：空气粒子原地振动、压缩区与稀疏区交替出现
- 教学层：顶部“课堂讲解要点”与底部字幕按时间轴同步切换

## 技术栈

- `Three.js`：3D 场景与动画
- `Vite`：开发服务器与构建工具
- 原生 `HTML/CSS/JS`：界面与交互

## 目录结构

```text
.
├── index.html              # 页面结构（控制面板、讲解区、字幕区）
├── src

│   └── style.css           # 样式
├── package.json            # 脚本与依赖
├── Sound-wave-mvp.html     # 早期单文件版本（保留参考）

```

## 本地运行

1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

3. 生产构建

```bash
npm run build
```

4. 预览构建结果

```bash
npm run preview
```

## 交互说明

- `播放/暂停`：控制动画时间推进
- `重置`：回到初始状态
- `音量`：影响振幅大小
- `频率`：影响振动速度
- 鼠标拖拽：旋转视角
- 鼠标滚轮：缩放视角

## 课堂演示设计要点

- 喇叭正对小球列，强调“声源 -> 受体”的传播方向
- 鼓膜做前后振动并有轻微呼吸缩放，突出“振动是源头”
- 小球响应满足“近处先动、振幅更大；远处后动、振幅衰减”
- 波前与粒子层同时显示，帮助学生建立宏观/微观对应关系

## 可调参数（代码中）

在 `src/main.js` 中可继续调优：

- 小球衰减曲线：`distanceAttenuation`
- 小球摆幅系数：`swing / zOffset / yLift` 的倍率
- 鼓膜振动幅度：`amp`
- 波前发射间隔：`spawnGap`
- 微观粒子振幅与密度表现：`amplitude`、`waveNumber`

## 已知说明

- `vite build` 可能提示主包体积超过 `500kB`（Three.js 单页场景常见），不影响运行。
